The script supports the following targets:

	dist: distribute all tools for all platforms.
	[platform].dist: distribute all tools for a specific [platform].
	[platform].dist-rp: distribute ripper for a specific [platform].
	[platform].dist-rl: distribute replayer for a specific [platform].

	release: release all tools for all platform.		
	release-[platform]: release all tools for a specific [platform].	
	
	clean: clean all builds

Currrent platforms supported:
	jfc: Java Swing 
	sel: Selenium 

Build a specific module:
	ant -f modules/[module name]/build.xml [target]
